# App Modernization Workshop

## Overview

This workshop will walk you through the process of modernizing a Java application using **GitHub Copilot app modernization**. You'll transform the `asset-manager` project from legacy technologies to a modern, cloud-ready solution.

**What the Modernization Process Will Do:**

The modernization will transform your application from outdated technologies to a modern solution. This includes:
- Upgrading from **Java 8 to Java 21**
- Migrating from **Spring Boot 2.x to 3.x**
- Adding **health check endpoints**
- **Containerizing** the applications
